🎨 Image Generation App

A side project using Stability AI's Stable Diffusion XL (SDXL)

🚀 Overview

This is a simple text-to-image generation application powered by Stable Diffusion XL (SDXL) Base 1.0 from Stability AI.
The app allows users to generate high-quality images from natural language prompts using state-of-the-art diffusion models.

✨ Features
🧠 AI-powered image generation from text prompts
⚡ Fast and responsive interface
🎯 High-quality outputs using SDXL Base 1.0
🖥️ Simple and clean UI
🔌 Easy integration with Stability AI API
🛠️ Tech Stack
Frontend: React.js, Tailwind CSS
Backend: Node.js / Express (or your backend if different)
API: Stability AI (Stable Diffusion XL Base 1.0)
📦 Installation
1. Clone the repository
git clone https://github.com/vrohan999/Image-Generation-App.git
cd Image-Generation-App
2. Install dependencies
npm install
3. Set up environment variables

Create a .env file and add:

VITE_HF_TOKEN=your_api_key_here
▶️ Running the App
npm start

Then open:

http://localhost:3000
🧪 Usage
Enter a text prompt (e.g., "A futuristic city at sunset, cyberpunk style")
Click Generate
View and download the generated image
📸 Example Prompts
"A dragon flying over a neon city"
"A realistic portrait of an astronaut in space"
"Fantasy landscape with glowing trees and waterfalls"
⚠️ Notes
Make sure your API key is valid and has sufficient credits
Image generation time may vary depending on server load
